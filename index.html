Welcome to the homepage of a collection of literate documents that explore the concept of relational programming, with a primary focus on miniKanren. Here, you'll find various implementations of miniKanren, optimization techniques for relational programming, useful tooling, and miscellaneous notes.

I haven't found a singular tool for doing literate programming that I like best. So I use two: Scribble works great for miniKanren embedded in Racket, because it can detect bound variables; for everything else, I use Literate.
